# <Gamers watch>

## サイト概要
一緒にゲームをプレイする仲間を募ることができるサイト。
フレンド同士でメールを使うこともできる。

### サイトテーマ
オンラインで遊べるゲームに特化している。

### テーマを選んだ理由
ゲームで遊んでいる友人がいても同じジャンルでなかったりプラットフォームが違ったりと、一緒に遊べる友人が少ないので仲間を募ることのできるものが
良いと思いこのテーマにしました。

### ターゲットユーザ
周りでゲームをやっている友人が少ない人、ゲーム仲間を増やしたい人。

### 主な利用シーン
遊びたいゲームで人数が足りていない時や固定で遊べる仲間を探している時など。
